## Topical Border

### Inside Core (must cover)
- All OT assignment types: case studies, reflective essays, literature reviews, activity analysis, research proposals, practice placement journals
- OT theoretical models as assignment subjects: MOHO, CMOP-E, OTPF, Kawa, OPM(A)
- All OT practice areas tested in degree programmes: paediatric, mental health, neurological, physical rehab, geriatric, community, hand therapy
- OT assessment tools as assignment content: COPM, AMPS, MOHOST, FIM, Barthel, Sensory Profile, MABC-2
- Condition-specific OT assignments: stroke, autism, dementia, CP, SCI, TBI, mental health
- Clinical reasoning types in written work
- Practice placement academic requirements (HCPC/RCOT framework)

### Inside Outer (must support)
- Occupational science foundations (occupation, justice, identity, deprivation)
- HCPC and RCOT professional standards
- Anatomy/physiology underpinning OT practice
- Disability models (social model, CRPD)
- NHS structure and MDT working context
- Assistive technology categories
- Global OT (WFOT, international programme differences)

### Outside Border (excluded)
- Physiotherapy assignment help
- Nursing assignment help
- Social work assignment help
- General healthcare essay writing
- Non-academic OT content

**Competitor threat:** Generic "healthcare assignment help" sites — maintain OT-exclusive vocabulary (MOHO, COPM, RCOT, occupational justice, activity analysis) across all N-grams.
