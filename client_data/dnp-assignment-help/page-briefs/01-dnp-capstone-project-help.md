> **EAV to cover**: practice vs research focus, methodology differences, capstone vs dissertation outputs
> **Word count**: 280–340 words
> **Specific content**:
> - DNP: practice doctorate — translates existing evidence into practice change; produces a practice improvement project or program evaluation

> - DNP capstone output: scholarly manuscript + practice change or program evaluation report

#### H2: The 8 Core Components of a DNP Capstone Project
> **EAV to cover**: all 8 components with brief definition of each
> **Word count**: 400–480 words
> **Specific content**:
> - **PICOT Question**: defines the clinical problem, population, intervention, comparison, outcome, and timeframe — the anchor of the entire project
> - **Literature Review**: systematic, scoping, or integrative review of 15–30 primary studies; establishes evidence base and identifies practice gap
> - **Theoretical/Conceptual Framework**: EBP framework (Iowa Model, PARIHS, ACE Star) or conceptual framework (Kotter, Rogers, Lewin) that guides the project design
> - **IRB Proposal**: determines whether the project qualifies as QI (non-research) or human subjects research; writes ethics documentation for the appropriate review level
> - **Implementation Plan**: details intervention rollout, stakeholder engagement, staff education, change management strategy, Gantt chart timeline, and fidelity monitoring
> - **Data Collection Tools**: instruments used to measure process and outcome variables — validated scales, audit tools, or survey instruments
> - **Evaluation Plan / Data Analysis**: pre-post comparison, statistical process control, descriptive statistics, or qualitative thematic analysis depending on project design
> - **Final Manuscript**: scholarly paper (60–120 pages) covering introduction, literature review, methods, results, discussion, implications for practice, and appendices

#### H2: DNP Project Types: QI, EBP, Program Evaluation, Policy, Guideline, and Educational
> **EAV to cover**: 6 project types with distinguishing characteristics
> **Word count**: 340–420 words
> **Specific content**:
> - **Quality Improvement (QI)**: uses IHI Model for Improvement or PDSA cycles; targets measurable clinical quality metrics (CAUTI rates, fall rates, readmission rates); primarily uses routinely collected EHR data
> - **EBP Implementation**: translates synthesised evidence (15–30 studies minimum) into a clinical practice change using a recognised EBP framework; requires formal literature synthesis as the driver
> - **Program Evaluation**: systematically assesses an existing clinical program or care delivery model; uses Logic Model, Donabedian, CIPP, or RE-AIM frameworks; common in education and leadership tracks
> - **Clinical Practice Guideline Development**: develops an evidence-based clinical protocol or care standard for a defined population; requires GRADE or JBI evidence appraisal
> - **Policy Change Initiative**: develops, advocates for, or evaluates a clinical or organisational policy; requires stakeholder analysis and organisational navigation
> - **Educational Program Development**: designs and evaluates a staff or patient education program; requires needs assessment, curriculum design, and learning outcome evaluation

### FAQ SECTION

**Q: What is a DNP capstone project?**
A DNP capstone project is a practice-focused doctoral scholarly project required to complete a Doctor of Nursing Practice degree. Unlike a PhD dissertation, it does not generate new primary research knowledge. Instead, it applies evidence-based practice, quality improvement, or program evaluation methods to a defined clinical or healthcare system problem — producing a sustainable practice change in a real clinical setting.

**Q: How is a DNP capstone different from a PhD dissertation?**
A DNP capstone focuses on translating existing evidence into practice change — it is a practice doctorate project. A PhD dissertation focuses on generating new knowledge through original research — it is a research doctorate requirement. DNP capstone students do not conduct primary research; they synthesise published evidence and apply it to improve clinical outcomes.

**Q: Which DNP specialisation tracks do you support?**
All 13 major DNP specialisation tracks are supported: FNP, AGACNP, AGPCNP, PMHNP, PNP, NNP, WHNP, CNM, CRNA, CNL, Nurse Executive, Population Health, and Nursing Informatics. Each track has distinct capstone topics, clinical settings, and outcome measurement requirements — all are addressed with track-specific expertise.

**Q: What DNP project types can you help with?**
All six major DNP project types are covered: Quality Improvement (QI), Evidence-Based Practice (EBP) Implementation, Program Evaluation, Clinical Practice Guideline Development, Policy Change Initiative, and Educational Program Development. The appropriate project type is determined by your clinical problem, available data, and university requirements.

**Q: How many components does a DNP capstone project have?**
A complete DNP capstone project has eight core components: PICOT question, literature review, theoretical/conceptual framework, IRB proposal, implementation plan, data collection tools, evaluation plan/data analysis, and final scholarly manuscript. Most universities structure these across a proposal (Chapters 1–3) and a final project report.
