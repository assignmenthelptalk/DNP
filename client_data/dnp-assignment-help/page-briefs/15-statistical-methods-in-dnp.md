## EAV Depth Map
**Primary Entity**: Statistical Methods for DNP Data Analysis

**Main Attributes**: descriptive statistics, pre-post tests, SPC methods, qualitative analysis, software, sample size, statistical vs clinical significance

**Derived Attributes**:
- **Descriptive Statistics**: frequencies (N, %), means (M), standard deviations (SD), medians, ranges, confidence intervals — sufficient for most small-sample QI projects; reported for all variables regardless of inferential testing
- **Pre-Post Tests**: paired t-test (continuous, normal, same sample), Wilcoxon signed-rank (continuous, non-normal or N<30), McNemar (categorical paired)
- **SPC Charts**: P-chart (proportions), U-chart (rates with varying denominators), I-MR (individual continuous), run chart (median-based, no control limits — suitable for N<20 data points)
- **Qualitative Methods**: thematic analysis (Braun & Clarke), content analysis (frequency-based)
- **Software**: SPSS (university license standard), Excel (basic descriptive + run charts), JASP (free alternative to SPSS with APA output), R (advanced, not required for DNP), NVivo (qualitative)
- **Sample Size**: most DNP QI projects: convenience sample, N=20–100; power calculation not always required; non-parametric tests preferred when N<30
- **Statistical vs Clinical Significance**: p<0.05 ≠ clinically meaningful; MCID for each outcome measure defines clinical significance

#### H2: Pre-Post Comparison Tests: Paired t-Test, Wilcoxon, and McNemar Explained
> Test selection decision tree: continuous/normal/N≥30 → paired t-test; continuous/non-normal or N<30 → Wilcoxon signed-rank; categorical/paired → McNemar. Assumption checks: Shapiro-Wilk for normality. APA reporting format for each test result.

> Chart type selection: proportions → P-chart; rates with varying denominators → U-chart; individual continuous → I-MR. Run chart as simpler alternative. Special cause rules. Minimum 20 data points recommendation for SPC control limits.

### FAQ SECTION

**Q: Do I need to run a power analysis for my DNP capstone project?**
A formal a priori power calculation is not always required for DNP QI projects that use convenience sampling. However, the sample size must be discussed in Chapter 3 of the proposal and in the limitations section of the manuscript. If the sample is small (N<30), non-parametric tests should be used, and the small sample should be acknowledged as a limitation when interpreting the results.

**Q: What is the easiest statistical software for DNP students?**
SPSS is the most commonly used statistical software in DNP programs, and most universities provide students with access through institutional licenses. For basic descriptive statistics and run charts, Microsoft Excel is sufficient and readily available. JASP is a free, open-source alternative to SPSS that produces APA-formatted output and is more user-friendly than R.
