## EAV Requirements

### Entity: Learning Needs Analysis (LNA)
- Performance gap: difference between current performance and required standard
- Role requirements: competencies and skills the role demands — from job description, competency framework
- Personal goals: career aspirations, CPD objectives, areas the individual wants to develop
- Sources of LNA data: appraisal records, manager observation, self-assessment, 360 feedback

### Entity: Honey and Mumford Learning Styles
- Activist: learns by doing — thrives in new experiences, role plays, hands-on activities
- Reflector: learns by observing — prefers time to review and analyse before acting
- Theorist: learns by thinking — prefers models, concepts, logical frameworks
- Pragmatist: learns by applying — prefers practical application, case studies, simulations
- Application: match learning method to dominant style for most effective outcome

### Entity: 70:20:10 Model
- 70%: on-the-job learning — stretch assignments, projects, new responsibilities
- 20%: social learning — mentoring, coaching, peer learning, feedback
- 10%: formal learning — courses, workshops, e-learning, qualifications
- Note: formal training is valuable but is only 10% of how adults actually learn

### Entity: Kirkpatrick's Four-Level Model
- Level 1 Reaction: how learners felt about the training — satisfaction surveys, feedback forms
- Level 2 Learning: what knowledge or skill was acquired — pre/post test, observation
- Level 3 Behaviour: whether the learning changed on-the-job behaviour — manager observation, performance data, 3-month review
- Level 4 Results: business impact — productivity, quality, retention, cost — measured against original performance gap
