- Question/brief type (single command verb question, multi-part assignment brief, work-based scenario)
- Answer depth (pass-level, merit-level, distinction-level response)
- Command verb compliance (Analyse, Evaluate, Discuss, Justify — each requires different answer depth)
- Levels covered (3, 4, 5, 6, 7)
- Turnaround (48-hour express, 5–7 day standard)

**Derived Attributes**:
- Brief analysis (writer reads and interprets the assignment brief before writing — not just answering literally)
- Learning Outcomes alignment (answer structured to address each LO specified in the unit)
- Assessment Criteria matching (answer meets each criterion stated in the marking rubric)

**Values per Attribute**:
- Answer depth at merit → analysis applied, multiple management theories cited, evidence integrated, command verb met at depth
- Answer depth at distinction → critical analysis, limitations of theories acknowledged, SMART recommendations, 15+ sources
- Turnaround → 48-hour for questions up to 2,000 words; 5–7 days for full assignments 3,000–5,000 words

**Attributes on THIS page**: Answer format types, command verb answer depth, Learning Outcomes alignment, level coverage, turnaround, ordering process.

**Attribute depth warning**: Do not process general assignment writing advice here — this page's macro-context is ANSWERS (direct, question-answering response) not the writing process. Students on this page want a solution, not guidance.
