## Executive Summary

The site is an early-stage topical authority build with strong positioning signals (multiple pages ranking positions 7–12) but critically low CTR. **88 total clicks from 5,646 impressions = 1.56% CTR.** The fundamental problem is not traffic volume — it is conversion failure at the SERP. The biggest single opportunity: the 5OS01 page receives 3,307 impressions at position 7.49 but generates only 0.24% CTR. Fixing this one page could nearly double total site clicks.

**Topical Authority Formula Applied:**
- Topical Coverage: ~35% (Level 5 partially, Level 3 sparse, Level 7 absent, no Outer Section)
- Historical Data: Accumulating since ~February 2026 (impression spike began Feb 26)
- Result: Low authority state — site ranks for isolated queries but not as a domain-level authority

## Section 2: Technical Issues (Fix First)

These issues are bleeding authority before any content is created. Fix in Phase 1 (Weeks 1–2).

### Issue 1: URL Typo — `/5oso3/` (CRITICAL)
- **Problem:** The URL uses the letter "O" instead of zero. Google is indexing a broken URL.
- **GSC Data:** 13 impressions at position 10.38, 0 clicks
- **Fix:** Create `/5os03-assignment-example/` with content. Set up 301 redirect from `/5oso3/` to the correct URL. Submit updated sitemap.

### Issue 2: Duplicate OS05 URLs (HIGH)
- **Problem:** Two separate URLs for the same module: `/5os05/` (5 impressions) and `/cipd-5os05-diversity-and-inclusion/` (3 clicks, 32 impressions). Google sees these as competing pages.
- **Fix:** Merge all content into one canonical URL: `/5os05-assignment-example/`. 301 redirect both old URLs to the canonical. Update all internal links.

### Issue 3: Trailing Slash Split — `/5hr01` vs `/5hr01/` (HIGH)
- **Problem:** Both URL variants exist, splitting PageRank and potentially confusing Google's canonical selection.
- **Fix:** 301 redirect `/5hr01` (no trailing slash) to `/5hr01/`. Or redirect both to `/5hr01-assignment-example/` if moving to the standardised convention.

### Issue 4: WordPress Author Archive Indexed (HIGH)
- **Problem:** `/author/jobmugachiagmail-com/` and `/author/jobmugachiagmail-com/page/2/` appear in SERPs. Position 22, 6 impressions. These pages confuse Google's topical understanding and expose personal information in the URL.
- **Fix:** In WordPress, go to Yoast SEO → Search Appearance → Archives → Author archives → set to noindex. Or use RankMath equivalent setting.

### Issue 5: Dead Level 5 Hub Page (MEDIUM)
- **Problem:** `/cipd-level-5-examples/` sits at position 63 with 1 impression and 0 clicks. The page exists but is not performing.
- **Fix:** Replace with `/cipd-level-5-assignment-examples/` (Phase 3). 301 redirect from old URL once new page is live and indexed.

## Section 3: High-Impact Optimisations (Weeks 3–4)

### Priority 1: Fix 5OS01 CTR (Target: 0.24% → 2%+)

**The problem:** 3,044 of the page's 3,307 impressions come from a single verbatim CIPD exam question about maternity leave and shared parental leave. The page ranks at position 7.29 for this query but generates 0 clicks. This is because:

1. The title tag does not mention maternity leave — students scan the SERP title and don't see their question reflected
2. The auto-generated SERP snippet is pulling overview text rather than the AC-specific passage
3. There is likely no dedicated H2 section that mirrors the verbatim question language

**Exact changes required:**

**Title tag (update to):**
```
5OS01 Specialist Employment Law Assignment Example | Maternity Leave, EU Law & More
```

**Meta description (update to):**
```
Worked example for CIPD 5OS01 Specialist Employment Law. Covers maternity leave, shared parental leave, EU employment law post-Brexit and employment regulation. All ACs answered.
```

**Content addition — add this H2 section:**
```
H2: The Major Ways in Which Maternity Leave and Shared Parental Leave Differ

[400-word substantive answer covering:
- Definition of maternity leave: statutory right, duration (52 weeks: 26 ordinary + 26 additional), SMP rates
- Definition of shared parental leave: what it is, eligibility conditions
- Key differences: who holds the right, flexibility, opt-in vs automatic entitlement,
  notice requirements, pay implications, how the weeks are divided
- Employer obligations for both
- Connection to the CIPD 5OS01 Assessment Criteria]
```

**Add near top of page — AC navigation table:**
```
| Assessment Criterion | Topic | Jump Link |
|---|---|---|
| AC 1.1 | [Topic] | [link] |
| AC 1.2 | [Topic] | [link] |
| AC X.X | Maternity leave and shared parental leave | #maternity-leave |
```

**FAQPage schema — add this entry:**
```json
{
  "@type": "Question",
  "name": "What are the major ways in which maternity leave and shared parental leave differ?",
  "acceptedAnswer": {
    "@type": "Answer",
    "text": "[Summary of the 400-word answer]"
  }
}
```

**Expected outcome:** Moving from 0.24% to 2% CTR on 3,044 impressions = ~61 additional clicks per quarter from this page alone.

### Priority 2: Fix 5CO01 Zero-Click Queries
