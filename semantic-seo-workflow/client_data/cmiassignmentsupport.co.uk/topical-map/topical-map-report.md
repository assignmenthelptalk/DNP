### Inside Core
- All 5 CMI qualification levels — Level 3 (First Line Management) through Level 7 (Strategic Management and Leadership)
- All qualification types — Award, Certificate, Diploma at each level
- All 80+ CMI units — L3: 301–312, L4: 401–411, L5: 501–525, L6: 601–616, L7: 701–717
- All assignment output formats — management report, essay, reflective account, case study, action plan
- All service delivery types — full writing, tutoring, coursework, resubmission, homework help, examples, answers
- CMI command verbs — Identify through Critically Analyse across the full cognitive hierarchy
- CMI assignment structure — executive summary, analysis sections, recommendations, Harvard referencing
- CMI assessment criteria — Pass/Merit/Distinction differentiation by level
- UK student profiles — NHS Band 7+, public sector managers, commercial managers, military officers

### Inside Outer
- CMI qualification framework — level navigation, Award/Certificate/Diploma structure, ACMI/MCMI/FCMI membership
- Management theories and frameworks used in CMI assignments (Tuckman, Gibbs, Kolb, Kotter, Porter, Bass and Avolio, etc.)
- Academic writing methodology — reflective practice, management report writing, Harvard referencing
- NHS management context — Band structure, NHS leadership development, CMI × NHS query cluster
- Qualification value and career ROI — CMI vs ILM, CMI vs MBA, employer recognition
- CMI assessment methodology — command verbs, marking criteria, format rules

### Outside Border (Excluded)
- ILM assignment help — different awarding body
- CIPD, ACCA, CIMA, CFA assignment help — different professional domain
- University degree or dissertation help — different academic context
- Generic management or leadership content with no CMI connection
