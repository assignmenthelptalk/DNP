##### H3: Command Verb Application — Discuss, Analyse, and Critically Analyse
> **Attribute + Value**:
> - Discuss → write paragraphs that present valid arguments for multiple positions → no single "correct answer" → conclusion draws reasoned position
> - Analyse → identify components of the entity → show how they relate → explain WHY, not just WHAT
> - Critically Analyse → analyse + identify weaknesses, gaps, limitations, and alternative interpretations → requires academic sources that challenge the primary view
> **Sequence modeling note**: Each command verb in subject position independently. Do not bundle them in one sentence.
> **Writer note**: This is a key differentiator — generic writers write descriptively. CMI-qualified writers understand the cognitive depth required by each command verb. Make this contrast explicit.

##### H3: Harvard Referencing in CMI Essays
> **Attribute + Value**: In-text citations in (Author, Year) format → full bibliography at end → minimum 6–8 sources → sources: academic journals, management textbooks, CMI publications, ManagementDirect
> **Writer note**: Students often lose marks for poor referencing. State explicitly that all essays are delivered with complete Harvard referencing included.
