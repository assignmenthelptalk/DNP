### H2: Brand Identity Components for Kenyan Businesses

**EAV depth required:**
- Entity: Brand Identity System (Kenya)
- Components:
  - Logo: primary mark, secondary/horizontal variant, icon/symbol only variant
  - Color palette: primary brand color + 1–2 secondary colors + neutral (white/grey/black) — defined in HEX, RGB, and Pantone (for print)
  - Typography system: primary font (for headings) + secondary font (for body text) — Google Fonts or licensed typefaces
  - Brand voice: formal or informal tone, key messaging phrases, tagline
  - Brand guidelines document: PDF specifying all of the above with usage rules — essential for maintaining consistency across web design, social media, and print
- What a logo alone is NOT: a logo without a brand guidelines document leads to inconsistent application — different shades of the brand color on different platforms, wrong fonts used by different vendors
- Brand vs logo distinction: the brand is the entire perception of your Kenyan business; the logo is one visual component of it

**Word count target:** 240–300 words

### H2: How Much Does a Logo Cost in Kenya?

### H2: Logo File Formats for Kenyan Business Websites

**EAV depth required:**
- SVG (Scalable Vector Graphic): vector format — scales to any size without pixelation, best for website use, smallest file size
- PNG with transparent background: for placement on different background colors — website headers, email signatures, social media
- WebP: for website use — smaller than PNG with same quality
- JPEG: NOT recommended for logos (compression artifacts)
- PDF: for print use (business cards, brochures, signage)
- AI/EPS: source files for designers to modify in future
- Common Kenyan business mistake: receiving a low-resolution JPEG from a designer and trying to use it on a website — results in blurry logo. Always request SVG source file.

**Word count target:** 200–250 words

### H2: Color Psychology for Kenyan Business Brands

**EAV depth required:**
- Kenya industry color conventions (what Kenyan customers expect):
  - Banking and finance: deep blue, gold — trust, stability (example: Equity Bank, KCB)
  - Healthcare: blue, green, white — health, cleanliness, trust
  - Legal services: navy, dark grey, gold — authority, formality
  - Hospitality: warm earth tones (safari orange, savannah brown), cream — warmth, Kenya nature
  - Education: blue, green — knowledge, growth
  - Technology/digital: electric blue, black, white — modernity, innovation
  - Construction/industrial: orange, grey — strength, engineering
- Rebranding consideration: established Kenyan businesses should maintain enough visual continuity that existing clients recognize the rebrand — sudden radical color changes can confuse and alienate existing Kenyan customers
- Color and digital consistency: brand color must be defined in HEX (for screens) — not just "dark blue" which varies between monitors and printers

**Word count target:** 220–280 words

### SUPP H2: Branding FAQs for Kenyan Businesses

**FAQ entries:**
1. Q: Should I have my branding done before my website? A: Yes — ideally. Website design that precedes branding results in a website that has to be redesigned when the brand is later formalized. Invest in basic brand identity (logo, colors, fonts) before approaching a Kenyan web designer.
2. Q: Can I use my existing logo for my new website? A: Yes — if you have an SVG or high-resolution PNG version. If your current logo exists only as a blurry JPEG or a screenshot, it will look unprofessional on a modern website and should be redesigned.
3. Q: Does Tupate Studio design logos? A: Yes — Tupate Studio provides logo design and basic brand identity packages for Kenyan businesses as part of website projects or as standalone engagements. We deliver SVG source files, brand guidelines PDF, and all web-ready file formats.
