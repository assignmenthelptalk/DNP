### H2: Heading Structure: How H1, H2, H3 Tags Affect Your Kenya Rankings

**Contextual directive:** Explain heading hierarchy as a semantic signal for Google — not just visual formatting. Show how wrong heading structure dilutes topic signals for Kenyan website pages.

**EAV depth required:**
- H1 rules: one per page, contains primary keyword, matches search intent exactly
- H2 rules: sub-topics that answer representative queries (not just cosmetic section labels)
- H3 rules: attribute-level depth under H2 context
- Common mistakes on Kenyan business websites: multiple H1s (common in WordPress themes), decorative headings (e.g., "Welcome to Our Company"), H3 used before H2
- Contextual hierarchy: the order of H2s changes the weight Google assigns to different topic aspects — the first H2 after H1 receives the most contextual weight
- Kenya example: a dental clinic page with H2 "Our Services" (vague) vs H2 "Dental Services in Nairobi: General Dentistry, Orthodontics, and Cosmetic Procedures" — the second version processes 3x more Kenyan search queries

**Word count target:** 300–360 words

### H2: Image Optimization for Kenyan Websites: Alt Text and WebP Format

**Contextual directive:** Cover image optimization specifically for Kenya's mobile-first, bandwidth-sensitive market. Connect image optimization to both SEO (alt text) and user experience (format/size).

**EAV depth required:**
- Alt text: descriptive, keyword-informed, specific ("custom website design for Nairobi law firm" not "image1.jpg")
- File format: WebP as default (30% smaller than JPEG at same quality — critical for Kenyan mobile data costs)
- File size targets: hero images max 120KB WebP, body images max 60KB, thumbnails max 20KB
- Lazy loading: defer off-screen images — reduces initial page load for Kenyan 4G users
- Descriptive file names: `/nairobi-website-designer.webp` not `/DSC00234.jpg`
- srcset attribute: serve smaller images to mobile screens (Tecno Spark resolution vs desktop) — reduces data consumption for Kenyan mobile users
- Image content: Kenya-relevant photography outperforms stock photos for trust signals on Kenyan business websites

**Word count target:** 280–340 words

### H2: Internal Linking Strategy: How Pages on Your Kenyan Site Support Each Other

**Contextual directive:** Explain internal linking as a contextual bridge mechanism — not just navigation. Show how it distributes PageRank and creates topical connections between Kenyan website pages.

**EAV depth required:**
- Internal link types: navigational (header/footer), contextual (within body content), supplementary (sidebar, related posts)
- Anchor text rules: descriptive, keyword-rich, matches the target page's H1 topic — never "click here" or "read more"
- Kenya context: linking "Google Business Profile optimization" anchor to the GBP service page from the Local SEO page — passes topical relevance AND PageRank
- Link depth: important service pages should be reachable within 2 clicks from homepage
- Over-linking warning: maximum 5 contextual internal links in main content — too many links dilutes the weight passed to each
- Orphan pages: pages with no internal links receive minimal crawl attention from Google — every new page on a Kenyan business website must receive at least 1 internal link from an existing page

### SUPP H2: On-Page SEO FAQs for Kenyan Businesses

**FAQ entries:**
1. Q: How long does on-page SEO take to show results in Kenya? A: On-page improvements are indexed by Google within 2–4 weeks for existing pages. Ranking improvements for competitive Kenyan keywords typically appear within 2–4 months of implementing full on-page optimization.
2. Q: Does changing my title tag affect my current rankings? A: Yes — title tag changes can cause ranking fluctuations for 2–6 weeks as Google re-evaluates the page. Always change title tags strategically, not frequently.
3. Q: How many keywords should one page target on a Kenyan website? A: One primary keyword (e.g., "dental clinic Nairobi") supported by 3–5 semantically related terms (e.g., "dentist Nairobi," "teeth cleaning Nairobi," "dental services Kenya"). One page = one clear topic.
4. Q: Is Swahili content needed for on-page SEO in Kenya? A: For businesses targeting Swahili-speaking Kenyan audiences, Swahili keywords ("bei ya website Kenya," "wataalamu wa SEO Nairobi") should appear in content — particularly in H2s and the first paragraph.
5. Q: What's the difference between on-page SEO and technical SEO? A: On-page SEO optimizes what's visible on the page (content, headings, images). Technical SEO optimizes server-side factors (speed, crawlability, indexation). Both are required for a Kenyan website to rank.
