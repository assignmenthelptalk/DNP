## The Custom-Built Positioning (Critical Brand Differentiator)

Tupae Studio builds custom-coded websites — not WordPress. This is not just a service detail. It is a **topical authority claim** that must change the macro semantics of the entire site.

**The dedicated page "Custom-Built Website vs WordPress Kenya" must cover:**
- Core Web Vitals comparison: custom code vs WordPress average page weight (2-4MB) for Kenyan 4G users
- Security: no plugin CVEs, no brute-force login — contrast with Kenya's high WordPress hack rate
- Speed: measurable LCP difference on Tecno/Infinix budget Android phones
- Total cost of ownership over 3 years (no license fees, no plugin renewals)
- SEO advantage: clean semantic HTML, precise schema, no conflicting plugin settings

**This page serves two purposes:**
1. Ranks for queries like "custom website Kenya", "hand-coded website Kenya" — captures high-intent buyers
2. Preemptively handles the sales objection "can't you just use WordPress?" for every client pitch
