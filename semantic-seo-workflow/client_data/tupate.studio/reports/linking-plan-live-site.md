## How to Use This Document

This is the complete internal linking specification for all 39 pages of tupate.studio.

**For each page:**
- Find the page on the live site
- Locate the **supplementary content section** (lower portion of the page, after the main content — the section that touches sub-topics and connects to other pages)
- Locate the **contextual border** (the transitional paragraph between main and supplementary content)
- Add the listed links by wrapping the specified anchor text with `<a href="[destination]">anchor text</a>`
- If the exact anchor text phrase doesn't exist verbatim, find the nearest natural equivalent in that paragraph and wrap it

**Placement key:**
- `SUPPLEMENTARY` = inside the supplementary content section
- `CONTEXTUAL BORDER` = inside the transitional paragraph between main and supplementary
- `MAIN CONTENT` = inside the main content section (only where specified — do not add to main content otherwise)

**Link format:** Root-relative. Use `/slug` not full URL. Example: `<a href="/seo-services-kenya">SEO Services in Kenya</a>`

**Do not:**
- Add a link if that destination URL already exists anywhere on the page
- Create a bullet list of links — all links must be embedded in existing sentence text
- Add more than 6 total internal links to any single page
