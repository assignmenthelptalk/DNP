#### H2: How to Read NEBOSH Command Words and Respond at the Right Level — *Major Query Need: command word strategy*
> **Context vector**: Command word compliance as the single most controllable pass/fail factor
> **Main entity + attributes to cover**:
> - Hierarchy: identify < outline < describe < explain < justify < evaluate (cognitive demand order)
> - "Explain" — the most commonly under-answered command word in NEBOSH; requires mechanism (how/why), not features (what)
> - Practical drill: take an "explain" question, write an "identify" answer, then expand it to "explain" level — show the difference
> - "Evaluate" — requires a conclusion/judgement; students who describe both sides without a conclusion lose marks
> - Marking allocation signal: "explain" questions carry more marks than "identify" questions; treat each mark as requiring a new distinct point
> **Predicate to use**: "requires" — "the command word 'explain' requires..."

#### H2: FAQ — First-Time Pass Representative Query Extensions

**Questions and Answer Directions**:
1. **Can I use my textbook during the NEBOSH open book exam?** → Answer: Yes. NEBOSH OBE is explicitly open-book — students may use their study materials, notes, and reference documents throughout the 24-hour window. However, copying text directly from study materials without application to the scenario will not earn marks. Examiners can identify generic textbook responses — apply the material to the question scenario.
2. **How long should a NEBOSH IG1 answer be?** → Answer: NEBOSH does not set a minimum word count but examiner guidance consistently indicates that most referred answers are too short on "explain" and "evaluate" tasks. A typical IG1 answer for a 10-mark question requires 3–4 well-developed paragraphs applying specific principles to the scenario. Quality of application outweighs quantity.
3. **Is it possible to get a distinction without expert help?** → Answer: Yes. Students who systematically review examiner reports, practise under timed conditions, and master command word compliance can achieve distinction independently. Expert help accelerates the learning by providing model answer benchmarks and targeted feedback on draft answers.
4. **What happens if I miss the NEBOSH OBE submission deadline?** → Answer: Missed submission within the 24-hour window is treated as a non-submission. The student must re-register for the next available OBE window, which involves a re-sit fee. NEBOSH does not accept late submissions for any reason after the window closes.
5. **How many times can I re-sit a NEBOSH assignment?** → Answer: NEBOSH permits re-submission of referred work. The specific number of permitted attempts and the time limit for re-submission are set by the approved learning provider in line with NEBOSH's qualification regulations. Most providers allow up to two further attempts.
