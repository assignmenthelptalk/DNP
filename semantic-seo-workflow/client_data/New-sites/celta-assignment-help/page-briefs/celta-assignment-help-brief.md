#### H2: Assignment 2 — Language-Related Tasks: The Most Technically Demanding Assignment
> **EAV to cover**: Assignment 2 entity × FMU+P framework, CCQs, language item types, Pass B requirements
> **Word count**: 300–380 words
> **Specific content**:
> - Purpose: to demonstrate that the candidate can fully analyse any language item (grammar structure, lexical item, or phonological feature) using the Form–Meaning–Use–Phonology (FMU+P) framework
> - Form: how the structure works in affirmative, negative, and interrogative sentences — using grammatical notation, naming auxiliary verbs, tense markers, and morphological features
> - Meaning: denotation (core meaning), connotation (associated meaning), appropriacy (who uses it and when)
> - Use: register (formal/informal/neutral), frequency, typical collocations, when this form is chosen over alternatives
> - Phonology: word stress (primary and secondary, marked with IPA stress marks), connected speech features (elision, assimilation, linking), full IPA transcription of the item in context
> - CCQs (Concept Checking Questions): simple questions to verify learner understanding without using the target language — 3–4 per item, binary or short-answer format
> - Most common failure: missing the phonology section entirely, or providing CCQs that use the target language in the question

> - Productive skills (speaking and writing): candidates analyse a communicative task for design quality, communicative purpose, and accuracy vs fluency balance
> - Lesson staging: pre-task (prediction activities, vocabulary pre-teaching, schema activation), while-task (graded comprehension questions targeting specific sub-skills), post-task production (speaking or writing activity connected to text theme)
> - Methodology: candidates must reference and justify their staging decisions using EFL methodology — PPP, TBL (Willis 1996), or ESA (Harmer)

### FAQ SECTION

**Q: What are the four CELTA written assignments?**
The four Cambridge CELTA written assignments are: (1) Focus on the Learner — a learner profile with error analysis and material recommendation; (2) Language-Related Tasks — a full language analysis using the Form, Meaning, Use, and Phonology (FMU+P) framework with CCQs; (3) Skills-Related Task — an analysis of a receptive or productive skills lesson with staged task design; and (4) Lessons from the Classroom — a critical reflective essay on teaching practice using Schon's reflection models and Kolb's Experiential Learning Cycle.

**Q: What is the difference between a Pass B and a standard Pass on CELTA?**
Pass B (~15–20% of candidates) requires consistently above-standard performance across written assignments and teaching practice. The distinction in written assignments is depth: a Pass B Assignment 2 includes complete FMU+P analysis with accurate IPA transcription and properly constructed CCQs; a Pass B Assignment 4 identifies patterns across multiple TP lessons rather than describing individual lessons. Standard Pass assignments meet all criteria without distinction-level analytical depth.

**Q: Can I get CELTA assignment help while still on the course?**
Expert guidance is available at any stage of the CELTA course — from pre-draft planning (understanding what the assignment requires before writing) to draft review (identifying gaps against Pass B criteria) to resubmission support (if an assignment has been referred). Guidance is tailored to the candidate's specific assignment context: their actual learner, chosen language item, or TP observations.

**Q: Which CELTA assignment do most candidates find hardest?**
Assignment 2 (Language-Related Tasks) is consistently the most technically demanding. It requires full language analysis using Form, Meaning, Use, and Phonology — including IPA transcription, connected speech features, and accurately constructed Concept Checking Questions. Assignment 4 (Lessons from the Classroom) is the most commonly written below standard because candidates default to describing lessons rather than analysing patterns in their development across the course.
