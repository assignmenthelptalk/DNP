## Sequencing Logic

**Phase 1 — Core Tier 1 (Service Foundation)**
Establish the main money pages first. These are the highest-intent, highest-conversion pages. Google must index these before any outer content links to them.

**Phase 2 — Core Tier 2 (Grading Intelligence + Subject Clusters)**
Grading criteria and strategy pages give the site its depth signal and differentiate it from thin competitors. Subject cluster pages expand Core coverage and begin feeding internal links into Tier 1 pages.

**Phase 3 — Outer Tier 1 (Qualification Discovery)**
Qualification overview and comparison pages — these capture students earlier in the journey and funnel them toward Core Section service pages.

**Phase 4 — Outer Tier 2 (Academic Process + Trust)**
Academic integrity, referencing, and assessment process pages build trust signals and create contextual bridges back to Core pages.

**Phase 5 — Outer Tier 3 + Gap Topics (Border Expansion)**
Contextual bridging pages, niche subject clusters, and gap topics that extend the topical border and strengthen historical data propagation.
